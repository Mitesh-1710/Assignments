CSS Selectors and Combinators

SELECTORS

1. Universal Selector :-

The universal selector (*) selects all HTML elements on the page.

Example:

    *{
        text-align: center;
        font-family: 'Times New Roman', Times, serif;
        color: peru;
        font-size: 30px;
    }

2. ID Selector :-

The #id selector styles the element with the specified id.

Example:

    #container{
        background-color: black;
        color: azure;
        padding: 20px;
    }

3. Class Selector :-

The .class selector selects elements with a specific class attribute.

Example:

    .nameBold{
        color: aliceblue;
        background-color: black;
        padding: 5px;
    }

4. Element Selector :-

The element selector selects all elements with the specified element name.

Example:

    p{
        color: blueviolet;
        background-color: rgb(70, 66, 66);
        font-size: 30px;
        font-family: 'Franklin Gothic Medium', 
                            'Arial Narrow', Arial, sans-serif;
        padding: 20px;
    }

5. Group Selector :-

To style several elements with the same style, separate each element name with a comma.

Example:

    h1 , p {
        text-align: center;
        color: rgb(36, 184, 134);
        background-color: azure;
        text-decoration: wavy;
        padding: 10px;
    }

COMBINATORS

1. Descendant Selector :-

The descendant selector matches all elements that are descendants of a specified element.

Example:

    div p{
        color: blanchedalmond;
        background-color: rgb(90, 88, 88);
    }
    
    p span{
        background-color: black;
        color: purple;
        padding: 5px;
    }

2. Child Selector :-

The child selector selects all elements that are the children of a specified element.

Example:

   
    div > h1{
        color: black;
        background-color: blueviolet;
        padding: 5px;
        text-decoration: underline;
    }

3. Adjacent Sibling Selector :-

The adjacent sibling selector is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" means "immediately following".

Example:

    div + p {
        color: purple;
        background-color: black;
    }

4. General Sibling Selector:-

The general sibling selector selects all elements that are next siblings of a specified element.

Example:

    div ~ h1{
        text-align: center;
        text-decoration: underline;
        border: 2px solid red;
        background-color: black;
        color: white;
    }
